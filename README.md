# IntelliTask AI - 智能作业管理系统

IntelliTask AI 是一款基于 Google Gemini AI 驱动的现代化智能教育辅助平台。它旨在通过 AI 技术打通作业从“布置”到“批改”再到“个性化提升”的全链路流程，帮助学生更高效地管理学业并精准修补知识漏洞。

## 1. 需求分析 (Requirement Analysis)

本项目核心解决以下三个教育痛点：

*   **作业采集碎片化**：家长和学生常需要从繁杂的 QQ/微信 群聊信息中手动筛选作业内容。
*   **反馈滞后**：传统人工批改周期长，学生无法在完成作业后的第一时间获得纠错反馈。
*   **盲目刷题**：学生往往不清楚自己的具体薄弱知识点，缺乏针对性的练习。

### 核心功能流程：
1.  **智能采集 (Inbox)**：实时监控并一键提取社交群聊中的作业任务（科目、内容、截止日期）。支持手动上传作业图片并利用 AI 提取任务细节。
2.  **AI 视觉批改 (Scanner)**：通过手机拍摄上传作业，AI 利用多模态视觉能力进行 OCR 识别、手写体逻辑分析及打分。支持自动触发与历史回顾。
3.  **缺陷诊断 (Diagnosis)**：自动生成知识点掌握程度报告，精准定位“哪里不会”。
4.  **闭环学习 (Learning Hub)**：基于诊断结果，AI 自动生成包含视频、练习及阅读材料的个性化提升计划。
5.  **成长看板 (Reports)**：通过可视化图表追踪长期学习趋势。

## 2. 技术架构解析 (Technical Architecture)

系统采用全栈式 AI-Native 架构，核心技术选型如下：

### 核心框架与语言
*   **React 19**: 利用组件化架构实现流畅的单页应用 (SPA) 体验。
*   **TypeScript**: 强类型系统确保了 AI 返回的复杂 JSON 数据在前端处理时的安全性和一致性。
*   **i18next-inspired Context API**: 自定义实现的轻量级国际化方案，支持多语言实时切换。

### AI 智能引擎
*   **Google Gemini API (@google/genai)**:
    *   **Gemini 3 Flash**: 用于轻量级的任务提取与快速 OCR，平衡响应速度与处理能力。
    *   **Gemini 3 Pro**: 负责核心的**视觉批改**与**深度逻辑推理**。它能识别复杂的数学公式、分析语境，并根据教学逻辑给出反馈。
    *   **Dynamic Language Prompting**: AI 服务层会根据当前 UI 语言（中/英）动态调整 Prompt。

### UI/UX 表现层
*   **Tailwind CSS**: 采用响应式设计，提供极简、专业的学术风界面。
*   **Lucide React**: 现代化的矢量图标，提升交互直观性。
*   **Recharts**: 实现动态的**学习趋势折线图**和**知识点掌握程度雷达图**。

## 3. 功能更新 (Feature Updates)

### [V1.1] 截止日期与手动管理
*   **截止日期集成**: 在作业录入环节引入了标准的日期选择器。
*   **手动录入模块**: 除了 AI 提取，用户现在可以通过“Add Manual”表单快速添加自定义任务。

### [V1.2] 多模态 AI 批改引擎
*   **OCR 文字提取**: 新增“Quick OCR”功能，利用 Gemini 3 Flash 实时提取作业原稿。
*   **Gemini Pro 深度批改**: 能够生成包含评分、详细反馈、及多维度知识点掌握度评分的综合报告。

### [V1.3] 作业分类与交互式诊断
*   **作业细分标签**: 引入了“重要考试”、“小测验”及“日常练习”分类。
*   **交互式分析**: 扫描报告中的知识点点击即可展开，实时展示诊断建议。

### [V1.4] 学习追踪与智能过滤
*   **可视化进度管理**: Learning Hub 新增动态进度环与任务核查机制。
*   **多维过滤器**: 仪表盘现支持按科目、类别及截止日期进行综合筛选。

### [V1.5] 全球化支持 (Internationalization)
*   **多语言切换 (EN/ZH)**: 顶部导航栏新增语言切换开关，支持中英文界面一键转换。
*   **AI 语言自适应**: AI 现在能够感知当前的界面语言，并自动以相应语言生成批改建议和学习计划。

### [V1.6] 图像持久化与自动批改 (Image Persistence & Auto-Grading)
*   **端到端图像流**: 在作业采集阶段上传的图片会自动持久化到任务对象中。
*   **智能触发**: 进入 AI 批改中心选择任务时，若图片已就绪，AI 将自动启动批改流程，大幅缩短操作路径。

### [V1.7] 批改历史记录管理 (Grading History Management)
*   **历史存证**: 批改中心新增“历史记录”标签页。
*   **无损回顾**: 用户无需重复上传或重新消耗 AI 生成配额，即可瞬间调取过往的批改报告、得分情况及原始作业底稿。

## 4. 本地构建与部署 (Local Build & Deployment)

1.  **准备配置文件**：在根目录创建 `.env` 文件：`API_KEY=你的_GEMINI_API_KEY`
2.  **启动服务 (Docker)**：`docker-compose up --build`
3.  **访问应用**：`http://localhost:8080`
